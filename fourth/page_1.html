프로젝트 1: 반응형 웹 포트폴리오

이 프로젝트는 개발자 겸 디자이너로서의 저의 역량을 보여주기 위해 제작된 반응형 웹 포트폴리오입니다. 사용자 친화적인 UI/UX 디자인과 최신 웹 기술을 결합하여, 다양한 디바이스에서 최적의 경험을 제공합니다.

사용 기술

  • HTML5
  • CSS3
  • JavaScript (ES6+)
  • Bootstrap 5
  • Figma (UI/UX Design)

주요 기능

  • 반응형 디자인 (데스크톱, 태블릿, 모바일 지원)
  • 부드러운 스크롤 및 섹션 이동
  • 블랙 앤 화이트 미니멀리즘 디자인
  • 자기소개, 기술 스택, 연락처 섹션

프로젝트 2: 온라인 쇼핑몰 백엔드 시스템

Python과 Django를 사용하여 개발된 온라인 쇼핑몰의 백엔드 시스템입니다. 사용자 인증, 상품 관리, 주문 처리, 결제 연동 등 핵심 기능을 구현했습니다.

사용 기술

  • Python
  • Django
  • REST Framework
  • PostgreSQL
  • Docker

주요 기능

  • 사용자 회원가입 및 로그인 (JWT 인증)
  • 상품 CRUD 및 검색 기능
  • 장바구니 및 주문 관리
  • 결제 API 연동 (Stripe 예시)
